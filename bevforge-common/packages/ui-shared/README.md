# ui-shared

Shared UI contracts and primitives used by all BevForge suites.

## Purpose
- Keep shell/navigation behavior consistent across OS/OPS/LAB/FLOW/CONNECT.
- Keep shared UX contract in one place for suite-specific threads.

## Current Contents
- `src/app-shell-contract.ts`: shared shell interface and required route metadata.
- `src/suite-theme-contract.ts`: shared base theme tokens and suite accent tokens.
